# ABSORPTION_REPORT: Superpowers (obra)

## Analysis Date: 2026-03-15
## Target: https://github.com/obra/superpowers

### 🏗️ Architecture
The `superpowers` framework is an agentic skills library and methodology designed to elevate AI agents (like Claude Code and Antigravity) to elite engineering levels. It centers on rigorous development patterns and autonomous subagent orchestration.

- **Skill-Based Modularity:** Capabilities are encapsulated as "skills" organized within a structured repository.
- **TDD-First Methodology:** Strict adherence to Test-Driven Development (Red/Green/Refactor).
- **Subagent-Driven Development:** Uses autonomous subagents for task execution, code review, and self-inspection.
- **YAGNI & DRY Principles:** Built-in safeguards against over-engineering and code duplication.

### 🔑 Key Components
- **Core Skills (`skills/`):** The engine of the framework, containing logic for planning, execution, and verification.
- **Testing Engine:** Specialized skills for async testing and anti-pattern detection.
- **Collaboration Layer:** Skills for brainstorming, Socratic design, and multi-agent coordination.
- **Meta-Skills:** Capabilities for agents to write and test their own new skills.

### 🚀 Extracted Skills
- **TDD Master:** Rigorous test-first implementation with automated verification loops.
- **Subagent Orchestrator:** Managing parallel streams of work via autonomous sub-specialists.
- **Socratic Architect:** Using brainstorming and deep questioning to refine system design before coding.
- **Meta-Skill Developer:** Ability to autonomously expand the agent's own capability set.

### 🛠️ Entry Points
- `skills/writing-plans`: The initial "Thought Expansion" phase.
- `skills/subagent-driven-development`: The core execution loop.
- `skills/requesting-code-review`: The elite verification gate.

### ⚠️ Risks & Bottlenecks
- **Context Density:** Highly technical methodology requires the agent to maintain deep state awareness.
- **Subagent Overhead:** Multi-agent coordination requires robust state management (provided by Antigravity's LangGraph/Metatron).
