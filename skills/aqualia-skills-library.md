# aqualia-skills-library

A repository of specialized agent skills and knowledge bases, including a SharePoint Audit Agent.

## Context
Use this skill library as a reference for quality-controlled agent skills. The current portfolio includes an MVP for SharePoint auditing.

## Key Features
- **SharePoint Audit Agent**: Specialized tool for auditing SharePoint environments.
- **Skill Repository**: A structured collection of skills with defined quality and security expectations.

## How to Use
1. **Audit**: Invoke the SharePoint Audit agent for workspace analysis.
2. **Reference**: Explore the library for patterns on creating secure and high-quality skills.
