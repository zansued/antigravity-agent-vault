# gemini-artilect-intelligence

An "Artificial Intellect" extension for Gemini CLI, designed to enhance the core analytical and reasoning capabilities of AI agents in complex environments.

## Context
Use this skill when performing high-level reasoning, multi-step planning, or complex problem-solving that requires a more "intellectual" and reflective agent state.

## Key Features
- **Artilect Persona**: Activates a specialized reasoning mode focused on deep analysis and architectural consistency.
- **Reasoning Chains**: Encourages the agent to maintain long, coherent chains of thought across multi-turn interactions.
- **Gemini CLI Integration**: Native extension for the gemini-cli framework.
- **Reflective Problem Solving**: Tools for the agent to question its own assumptions and verify solutions more rigorously.

## How to Use
1. **Activate**: "Switch to Artilect mode to analyze the [ComplexIssue] architecture".
2. **Deep Dive**: "Perform an Artilect-level reasoning cycle on the proposed [DesignChange]".
3. **Verify**: "Use Artilect-reasoning to verify the correctness of the latest [Algorithm] implementation".

## Requirements
- **Runtime**: Gemini CLI.
- **Infrastructure**: Access to high-capability Gemini models (e.g., 1.5 Pro).
