# notion-agent-extension

An enhanced Notion integration for AI agents, providing 46 comprehensive tools for document automation, database management, and project tracking.

## Context
Use this skill to fully automate your Notion workspace, from simple page edits to complex database workflows and document generation.

## Key Features
- **46 Native Tools**: Includes 38 core Notion operations and 8 enhanced multi-step tools.
- **Document Automation**: Specialized workflows for generating reports, project briefs, and meeting notes.
- **Project Management**: Tools for managing databases, status updates, and task dependencies within Notion.
- **Analytics & Monitoring**: Enhanced tools for monitoring workspace activity and generating analytics.
- **Cache Support**: Uses a local cache (`.notion-cache.json`) for high-performance metadata lookups.

## How to Use
1. **Automate**: "Generate a project roadmap in Notion based on the current implementation plan".
2. **Manage**: "Add a new task to the [DatabaseName] and set the priority to High".
3. **Query**: "List all pages shared with the current integration in the [Workspace]".

## Requirements
- **API**: Requires a Notion Internal Integration Token.
- **Runtime**: Gemini CLI or MCP-compliant agent.
