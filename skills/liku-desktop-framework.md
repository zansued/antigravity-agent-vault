# liku-desktop-framework

A cross-platform desktop application framework using JavaScript, HTML, and CSS (Electron-based) for rapid app development and distribution.

## Context
Use this skill to scaffold, build, and package desktop applications that leverage web technologies while maintaining native system access.

## Key Features
- **Cross-Platform**: Build once for Windows, macOS, and Linux.
- **Web Tech Core**: Uses standard JavaScript, HTML, and CSS for UI implementation.
- **Native Access**: Full access to the underlying operating system via Electron's API.
- **Simplified Scaffolding**: Rapidly set up new desktop project structures.
- **Package Management**: Integrated tools for bundling and distributing desktop apps.

## How to Use
1. **Scaffold**: "Initialize a new Liku desktop project for the [AppName] application".
2. **Build**: "Build the production-ready distribution of the current Liku app for Windows".
3. **Debug**: "Launch the Liku developer environment to inspect the Electron renderer process".

## Requirements
- **Tools**: Node.js/NPM.
- **Integration**: Works seamlessly with Antigravity and other Node-based agent environments.
