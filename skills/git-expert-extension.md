# git-expert-extension

A specialized Gemini CLI extension designed to provide expert-level Git assistance, including advanced diff analysis, commit message generation, and workflow optimization.

## Context
Use this skill when you need a deep audit of git history, complex merging assistance, or automated generation of high-quality commit messages based on staged changes.

## Key Features
- **Git Audit**: Analyzes repository history to identify patterns, authorship details, and potential issues.
- **Commit Generator**: Generates descriptive, conventional commit messages by analyzing the content of staged diffs.
- **Diff Specialist**: Provides high-level summaries and detailed explanations of complex git diffs.
- **Workflow Optimization**: Recommends best practices for branching, merging, and maintaining clean repository history.
- **Agentic Helper**: Specifically structured to assist AI agents in managing local Git state with higher precision.

## How to Use
1. **Analyze**: "Audit the git history of the [BranchName] and identify the main contributors and change patterns".
2. **Commit**: "Generate a conventional commit message for the currently staged changes in the repository".
3. **Summarize**: "Explain the key architectural changes introduced in the diff between [CommitA] and [CommitB]".

## Requirements
- **Tools**: Local git installed.
- **Runtime**: Gemini CLI + Git Expert Extension.
