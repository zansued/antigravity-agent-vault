# neo-skills

A comprehensive suite of multi-domain expert skills for Gemini CLI, enabling "Perception-Reasoning-Action" loops.

## Context
Use this skill for a wide range of development tasks, from infrastructure design to code review and security auditing.

## Built-in Skills
- **Azure Pipelines Architect**: Specialized in CI/CD design for Azure.
- **Smart Git Assistant**: advanced git workflow automation.
- **Technical Architect**: System design and architectural planning.
- **Code Review Expert**: In-depth analysis of code quality and patterns.
- **Docker Generator**: Automated generation of Docker/Container configs.
- **Security Guard**: Enforces security best practices and identifies vulnerabilities.

## How to Use
1. **Design**: "Act as an Azure Pipelines Architect and design our CI/CD".
2. **Review**: "Review this pull request for security issues using Security Guard".
3. **Automate**: Use the Git assistant for complex branch management.

## Requirements
- **Platform**: Designed specifically for Gemini CLI (MCP-based).
