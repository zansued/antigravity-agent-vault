# aiqbee-ai

Connect AI assistants to your Aiqbee knowledge graph via MCP for brain discovery and management.

## Context
Use this skill when you need to interact with an Aiqbee knowledge graph (KG) to retrieve entities (neurons), relationships, and perform complex information traversal.

## Tools
- **Brain Discovery**: Discover and manage brain (KG) instances.
- **Neurons & Relationships**: Create, update, and query neurons and the relationships between them.
- **Search & Retrieval**: Advanced search within the knowledge graph for context recovery.
- **Access Control**: Manage permissions for KG assets.

## How to Use
1. **Init**: Connect via `npx -y aiqbee-ai`.
2. **Explore**: Use brain discovery tools to identify relevant knowledge silos.
3. **Query**: Retrieve specific neurons or relationships to augment the AI's context.

## Requirements
- **Authentication**: Aiqbee API credentials.
- **MCP Client**: compatible client like Claude Desktop or Gemini CLI.
