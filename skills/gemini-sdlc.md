# gemini-sdlc

A comprehensive SDLC Command Center for automated software development lifecycle management.

## Context
Use this skill to automate the entire software development lifecycle, from requirements gathering to maintenance and monitoring.

## Tools
- **Discovery/Planning**: Automate initial project scoping.
- **Requirements**: Generate and manage technical requirements.
- **Design/Architecture**: Assist in system design and architectural mapping.
- **Dev/Test/Deploy**: Automate coding tasks, test execution, and deployment triggers.
- **Monitoring/Maintenance**: Tools for ongoing system health checks and updates.

## How to Use
1. **Scoping**: Start with the discovery tools for a new project.
2. **Pipeline**: Move through the SDLC stages using targeted commands.
3. **Automate**: Use the utility tools for general SDLC automation tasks.

## Requirements
- **Environment**: Optimized for Gemini-based workflows.
