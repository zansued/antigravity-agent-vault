# supreme-power-core-skills

A comprehensive library of core skills and interaction patterns designed to extend the base capabilities of AI agents (Claude Code / Gemini CLI).

## Context
Use this skill when you need standardized, robust implementations of common agentic operations or when bootstrapping a new agentic workspace with advanced system tools.

## Key Features
- **Core Extension Library**: A curated collection of "superpowers" for system interaction and task handling.
- **Interaction Patterns**: Standardized methods for agent-user and agent-system communication.
- **Phase-Based Engine**: Implements the SupremePower Active Engine (Phase 3) logic for task execution.
- **Modular Design**: Tools are decoupled and can be selectively integrated into existing workflows.
- **Enhanced System Access**: Provides refined primitives for file management, search, and command execution.

## How to Use
1. **Bootstrap**: "Integrate the SupremePower core skills into the current agent configuration".
2. **Execute**: "Use the [SkillName] from the superpowers library to perform the [Action]".
3. **Extend**: "Develop a new system tool following the SupremePower Active Engine patterns".

## Requirements
- **Runtime**: Claude Code / Gemini CLI.
- **Environment**: Compatible with Node.js based agentic environments.
