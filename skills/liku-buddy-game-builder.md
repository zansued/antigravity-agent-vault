# liku-buddy-game-builder

An AI-powered game builder and management hub extension, featuring a "Buddy System" for agent-assisted game development and real-time LikuOS stats.

## Context
Use this skill to build, manage, and play community-driven terminal games, leveraging AI assistance for game creation and real-time performance monitoring.

## Key Features
- **AI-Powered Game Builder**: Assisted tools for creating new terminal games using LLMs.
- **Game Hub**: Centralized access point for built-in and community-developed games.
- **LikuOS Real-Time Stats**: Provides JSON-formatted real-time statistics for game sessions and agent performance.
- **Buddy System**: Intelligent assistant (Buddy) integrated into the building and playing workflows.
- **Persistence & Settings**: Automatic management of user preferences and game progress.

## How to Use
1. **Build**: "Use LikuBuddy to generate a new terminal-based [GameType] game".
2. **Stats**: "Get the current LikuOS real-time stats for the active session".
3. **Manage**: "Review the available community games in the LikuBuddy hub".

## Requirements
- **Runtime**: Node.js/NPM + Liku Desktop Framework (recommended).
- **Communication**: Supports JSON-based state exchange for AI agents.
