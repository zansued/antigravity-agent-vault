# atxp-agent-payment-infrastructure

ATXP (Agent Payment & Wallet Infrastructure) provides AI agents with essential identity and financial primitives, including a wallet, email, phone number, and access to paid services.

## Context
Use this skill when an agent needs to perform financial transactions, send/receive SMS, make voice calls, or access premium pay-per-use AI tools.

## Key Features
- **Agent Wallet**: Integrated financial wallet allowing agents to manage funds and pay for services autonomously.
- **Agent Identity**: Provides the agent with a dedicated email address and phone number for external communication.
- **Communication Suite**: Primitives for sending SMS messages and initiating AI-powered voice calls.
- **Paid Tool Access**: Instant access to premium MCP tools for web search, image/music generation, and code execution.
- **Self-Registration**: Automated infrastructure for agent self-registration and account management.

## How to Use
1. **Wallet**: "Check my ATXP wallet balance and report the last 5 transactions".
2. **Call**: "Initiate a voice call to [PhoneNumber] and deliver the [MessageContent]".
3. **Generate**: "Access the ATXP premium music generation tool to create a track for [Description]".

## Requirements
- **Infrastructure**: ATXP account or automated self-registration enabled.
- **Runtime**: Gemini CLI + ATXP MCP extension.
