# Squad Orchestrator (Nirvana Multi-Agent)

Gerencia a colaboração harmoniosa entre múltiplos agentes para resolver problemas complexos que extrapolam a capacidade de um único loop de execução.

## Papéis do Esquadrão
- **The Architect**: Define a estrutura estratégica e o design de interfaces.
- **The Grunt (Coder)**: Implementa o código atômico seguindo os Blueprints.
- **The Sentinel (Reviewer)**: Valida segurança, performance e conformidade com o framework CELF.
- **The Watchdog**: Monitora falhas, loops infinitos e 'alucinações de squad'.

## Padrões de Orquestração
- **Autopoietic Loop**: O sistema detecta uma lacuna de skill e spawna um agente para criar essa nova capacidade.
- **TDD Universe**: Ciclos automáticos de `Teste -> Falha -> Implementação -> Refatoração` entre diferentes agentes.
- **Collective Intelligence**: Fusão de insights de múltiplos modelos (Gemini + Claude + GPT) para validação cruzada.
