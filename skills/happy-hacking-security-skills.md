# happy-hacking-security-skills

A collection of agent skills specialized in interacting with open-source security tools and performing security audits.

## Context
Use this skill for automated security testing, vulnerability scanning, and auditing codebases for common security flaws.

## Key Features
- **Security Tools Integration**: Wrappers for popular OSINT and vulnerability scanners.
- **Audit Workflows**: Standardized loops for identifying security risks in code.
- **Multi-Agent Support**: Compatible with Claude Code, Codex, and Gemini CLI.
- **Marketplace Ready**: Designed for easy distribution and modular skill loading.

## How to Use
1. **Scan**: "Perform a security audit of the current repository using Happy Hacking skills".
2. **Audit**: "Evaluate the [dependency] for known vulnerabilities".
3. **Report**: Generate structured security findings for the development team.

## Requirements
- **Tools**: May require specific security tools installed in the environment (e.g., bandit, semgrep).
