# r-package-dev-gemini

A Gemini CLI extension for R package development following S7, devtools, and CRAN best practices.

## Context
Use this skill for standard R package development focusing on modern S7 OOP, devtools workflows, and CRAN compliance.

## Command Categories
- **Setup & Planning**: Project initialization and blueprinting.
- **Code Quality**: Automated linting and quality checks (CRAN-style).
- **Code Development**: Tools for S7 object-oriented programming in R.
- **Documentation**: Automated help files and pkgdown integration.
- **Release**: Workflow tools for preparing and submitting to CRAN.

## How to Use
1. **Workflow**: Follow the `devtools` standard (Load -> Document -> Test -> Check).
2. **S7**: Use specific commands for creating and managing S7 classes.
3. **Check**: Run `CRAN Check` simulations before release.

## Requirements
- **R Packages**: Requires `devtools`, `S7`, and `pkgdown`.
- **Environment**: Gemini CLI.
