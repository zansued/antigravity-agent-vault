# nano-banana-pro-skills

A collection of high-performance, modular skills ("Nano Banana") designed for advanced automation within the Gemini ecosystem.

## Context
Use this skill to augment your agent with a diverse set of "pro" utilities designed for speed and modularity in varied automation tasks.

## Key Features
- **Modular Utilities**: A wide range of specialized skills that can be combined for complex workflows.
- **Gemini App Integration**: Optimized for use within the standard Gemini App interface and CLI.
- **High Performance**: Designed with a "nano" philosophy for low-latency execution and minimal overhead.
- **Pro-Level Automation**: Focuses on tasks that require higher precision than standard agent out-of-the-box capabilities.

## How to Use
1. **Execute**: "Run the Nano Banana [SkillName] to process the [TargetData]".
2. **Chain**: "Combine the [SkillA] and [SkillB] utilities to automate the full [Workflow]".
3. **Inspect**: "List all available Nano Banana pro skills in the current registry".

## Requirements
- **Runtime**: Gemini CLI or Gemini App.
- **Architecture**: Supports various transport methods (MCP, native extensions).
