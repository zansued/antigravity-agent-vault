# science-skills

Scientific tools and chemistry knowledge for AI agents.

## Context
Use this skill for science-related tasks, specifically in chemistry, element lookup, and formula execution.

## Key Features
- **Chemistry**: Information on chemical elements, properties, and reactions.
- **Formula Execution**: Tools to execute or validate scientific formulas.
- **Element Info**: Detailed data retrieval for the periodic table.

## How to Use
1. **Lookup**: "Get element information for Gold".
2. **Formula**: "Execute the formula for [specific reaction]".
3. **Learn**: Ask questions about chemistry topics supported by the extension.
