# shopify-dev-mcp-extension

An official-style Shopify development extension for Gemini CLI, pre-configured for Shopify-specific engineering workflows and MCP transport.

## Context
Use this skill when developing Shopify apps, themes, or custom storefronts to leverage specialized Shopify development tools and context.

## Key Features
- **Shopify Dev MCP Integration**: Connects Gemini CLI with Shopify's internal development tools.
- **Workflow Optimization**: Tailored for Shopify's engineering standards and project structures.
- **Resource Management**: Tools for managing Shopify resources (products, orders, themes) via agentic commands.
- **Secure Transport**: Uses standardized MCP protocols for reliable communication with Shopify services.

## How to Use
1. **Setup**: "Configure the Shopify Dev MCP environment for the current project".
2. **Develop**: "Generate a Shopify [ResourceType] scaffold following best practices".
3. **Query**: "List all active themes in the connected Shopify store".

## Requirements
- **Tools**: Shopify CLI installed and authenticated.
- **Runtime**: Gemini CLI + Node.js.
