# mcp-amr

Model Context Protocol (MCP) server for Azure Managed Redis (AMR).

## Context
Use this skill to manage and interact with Azure Managed Redis instances directly through your AI agent.

## Tools
- **AmrTools**: A suite of tools for Redis data manipulation and instance management.
- **Azure Integration**: Seamlessly connects to Azure-hosted Redis services.

## How to Use
1. **Host**: Can be hosted remotely on Azure or run locally.
2. **Config**: Requires Azure credentials and Redis connection string.
3. **Query**: Perform SET, GET, and other Redis operations via natural language.

## Requirements
- **Cloud**: Azure account with Managed Redis instance.
