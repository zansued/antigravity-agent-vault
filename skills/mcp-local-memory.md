# mcp-local-memory

A lightweight, powerful local memory server for AI agents supporting entities, relations, and shared context.

## Context
Use this skill to give agents long-term persistence, cross-agent memory, and a structured understanding of users and codebases.

## Tools
- **Memory Management**: Create, retrieve, and delete text-based memories.
- **Knowledge Graph**: Manage entities and the complex relationships between them.
- **Task Management**: Track agentic tasks and their historical context.
- **Retrospective Extraction**: Automatically extract learnings from past interactions.

## Key Features
- **Shared Context**: Multiple agents can access the same local memory pool.
- **Natural Evolution**: Memory evolves and refines over time without manual pruning.
- **Async Operations**: Optimized for non-blocking memory reads/writes.

## How to Use
1. **Init**: Connect via `npx -y mcp-local-memory`.
2. **Store**: "Remember that the user prefers glassmorphism in UI".
3. **Recall**: "What are the established patterns for our database schema?".

## Requirements
- **Runtime**: Node.js v20+.
