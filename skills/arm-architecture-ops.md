# arm-architecture-ops

The official Arm MCP Gemini CLI Extension, enabling agents to interact with Arm-specific development tools, architecture documentation, and optimization primitives.

## Context
Use this skill when developing for Arm architectures, optimizing software for Arm-based systems, or querying Arm technical documentation.

## Key Features
- **Architecture Insights**: Direct access to Arm architecture specifications and optimization guides via MCP.
- **Toolchain Integration**: Primitives for interacting with Arm development tools and compilers (GCC, LLVM Arm plugins).
- **Optimization Primitives**: Guides the agent in applying Arm-specific performance optimizations and code transformations.
- **Development Workflow**: Streamlines the lifecycle of Arm-targeted software development within the Gemini CLI.
- **Direct Documentation Access**: Maps complex architectural concepts to agentic queries for rapid knowledge retrieval.

## How to Use
1. **Optimize**: "Apply Arm-specific optimization strategies to the performance-critical path in [SourceFile]".
2. **Query**: "Search the Arm architecture documentation for details on [Instruction/Feature]".
3. **Build**: "Configure the local toolchain for Arm-targeted cross-compilation of the [ProjectName]".

## Requirements
- **Tools**: Arm development toolchain (optional but recommended for full feature set).
- **Runtime**: Gemini CLI + Arm MCP Extension.
