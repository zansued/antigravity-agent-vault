# accessibility-agents

A specialized suite of 11 accessibility experts dedicated to enforcing WCAG 2.2 AA compliance in AI-generated code.

## Context
Use this skill to ensure that all generated UI components and documents meet international accessibility standards.

## Specialist Team
- **WCAG Enforcers**: Verify compliance against WCAG 2.2 AA criteria.
- **Review Agents**: Specialized agents for Claude Code, Copilot, and Gemini.
- **Workflow Agents**: Integrated into GitHub workflows for continuous accessibility auditing.

## Key Features
- **One-Click Install**: Fast deployment via standard npm commands.
- **No-Overwrite Install**: Safe installation that doesn't conflict with existing project files.
- **Thinking Phrases**: Customizable prompts for deeper accessibility reasoning in VS Code.

## How to Use
1. **Audit**: "Run an accessibility review on this React component".
2. **Fix**: "Modify this HTML to meet WCAG 2.2 AA standards".
3. **Continuous**: Integrate into GitHub actions for automatic PR auditing.

## Requirements
- **WCAG Standard**: Target WCAG 2.2 Level AA.
