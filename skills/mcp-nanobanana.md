# mcp-nanobanana

MCP-based image generation extension using Nano Banana and OpenRouter.

## Context
Use this skill when you need to generate images, icons, patterns, or technical diagrams directly within your AI agent's workflow.

## Key Features
- **Image Generation**: High-quality image creation via Nano Banana (Flux/SDXL etc. via OpenRouter).
- **Icon & Texture Generation**: Specific tools for creating UI icons and repeatable textures.
- **Technical Diagrams**: Automated generation of technical schematics and diagrams.
- **Visual Storytelling**: Tools to assist in sequential image generation for stories.
- **File Management**: Smart filename generation and duplicate prevention.

## How to Use
1. **API Key**: Configure `OPENROUTER_API_KEY`.
2. **Command**: Use specific commands for icons, patterns, or general images.
3. **Prompt**: supports natural language descriptions and advanced generation options (aspect ratio, focus, style).

## Requirements
- **API Key**: OpenRouter.ai account.
- **MCP Client**: compatible with the standard Model Context Protocol.
