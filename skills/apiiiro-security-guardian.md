# apiiro-security-guardian

A security-focused Gemini CLI extension by Apiiro, providing deep visibility into application security posture, risk analysis, and software supply chain security within the agent's workflow.

## Context
Use this skill when auditing codebases for security vulnerabilities, assessing supply chain risks, or integrating security-first thinking into the development lifecycle.

## Key Features
- **Guardian Agent**: An AI-native security assistant that monitors and alerts on security risks in real-time.
- **Risk Analysis**: Performs deep analysis of code and dependencies to identify critical security gaps.
- **Supply Chain Security**: Audits the software supply chain to detect malicious components or compromised dependencies.
- **Security Posture Visibility**: Provides a comprehensive dashboard and CLI summaries of the project's overall security health.
- **Automated Remediation**: Suggests and guides the agent through security fixes and mitigation strategies.

## How to Use
1. **Audit**: "Run a full security audit of the current project using the Apiiro Guardian Agent".
2. **Scan**: "Scan the software supply chain for any high-risk dependencies in the current workspace".
3. **Fix**: "Identify the most critical security risk and help me implement a remediation plan".

## Requirements
- **Account**: Apiiro account with API access enabled.
- **Runtime**: Gemini CLI + Apiiro MCP Server.
