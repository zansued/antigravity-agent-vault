# agent-skills-toolkit

A collection of polished and personal skills for AI agents, featuring a centralized skills-dashboard and a variety of command-based utilities.

## Context
Use this skill to enhance your agent's utility toolset with community-polished commands and to manage these skills through a visual dashboard.

## Key Features
- **Polished Skills**: A set of well-maintained and tested skills for common agent tasks.
- **Skills Dashboard**: A visual interface for monitoring, configuring, and testing agent skills.
- **Personal Command Set**: Extensible collection of `cmd-*` style utilities for custom automation.
- **Modular Design**: Easy to plug into existing agent architectures (Antigravity, Gemini CLI, etc.).

## How to Use
1. **Dashboard**: "Launch the agent-skills dashboard to review available utilities".
2. **Execute**: "Run [SkillName] to perform the requested automation".
3. **Configure**: "Update the configuration for the [cmd-Utility] within the toolkit".

## Requirements
- **Runtime**: Node.js/NPM.
- **Integration**: Supports various agent frameworks via standardized skill interfaces.
