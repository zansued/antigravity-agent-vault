# agentic-skill-template

A standardized scaffold for creating and distributing new agentic skills within the Metatron ecosystem.

## Context
Use this skill when you need to bootstrap a new skill from scratch, ensuring it follows established patterns for perception, reasoning, and action.

## Key Components
- **Perception Layer**: Template for gathering context from the environment.
- **Reasoning Loop**: Structure for defining agentic decision-making logic.
- **Action Schema**: Standardized format for tool and command definitions.
- **Documentation**: Includes a pre-formatted `SKILL.md` template.

## How to Use
1. **Clone**: Use this as the base for any new skill in the `skills/` directory.
2. **Configure**: Update the YAML frontmatter with the skill name and description.
3. **Implement**: Fill in the core logic following the template's placeholders.

## Requirements
- **Structure**: Must follow the Metatron Vault skill directory format.
