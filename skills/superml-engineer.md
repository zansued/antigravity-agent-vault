# superml-engineer

An ML engineering plugin for AI coding agents, providing expert-level capabilities for model development, training, and optimization.

## Context
Use this skill when working on machine learning projects, including model selection, training script generation, data preprocessing, and performance tuning.

## Key Features
- **ML Lifecycle Support**: Assisted model training, evaluation, and deployment workflows.
- **Optimization Skills**: Tools for hyperparameter tuning, model quantization, and pruning.
- **Data Engineering**: Specialized skills for handling massive datasets and creating robust preprocessing pipelines.
- **Enterprise Ready**: Features specifically designed for production-grade ML infrastructure.
- **Cross-Agent Parity**: Compatible with Antigravity, Gemini CLI, Claude Code, and Codex.

## How to Use
1. **Develop**: "Help me write a training script for a [ModelType] using SuperML best practices".
2. **Optimize**: "Optimize the current model for low-latency inference on [TargetDevice]".
3. **Analyze**: "Analyze the training logs and suggest improvements to the loss function".

## Requirements
- **Frameworks**: PyTorch, TensorFlow, Scikit-learn support.
- **Runtime**: Remote MCP or local install.
