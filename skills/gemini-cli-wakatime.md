# gemini-cli-wakatime

Integration for tracking AI-assisted coding activity and productivity via WakaTime/Wakapi.

## Context
Use this skill to monitor and analyze the time spent prompted AI, lines of code generated, and general development activity within Gemini CLI.

## Key Features
- **Time Tracking**: Monitors active prompting and execution time.
- **Activity Metrics**: Tracks LoC (Lines of Code) generated by the AI agent.
- **Dashboard Sync**: Seamlessly pushes metrics to WakaTime or private Wakapi instances.
- **Extension Management**: Integrated tools for listing, updating, and verifying the extension status.

## How to Use
1. **Config**: Set your `WAKATIME_API_KEY`.
2. **Track**: The extension runs in the background during Gemini CLI sessions.
3. **Analyze**: View metrics on your WakaTime dashboard to assess AI productivity.

## Requirements
- **Vault**: WakaTime or Wakapi account.
- **Environment**: Gemini CLI.
