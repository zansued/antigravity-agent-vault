# browserbase-local

MCP server for Browserbase, leveraging Stagehand v3 for advanced browser automation and stealth.

## Context
Use this skill for highly reliable browser automation that requires stealth, proxy support, or specific viewport configurations. Optimized for local and cloud-based Browserbase sessions.

## Key Features
- **Stagehand v3**: Uses the latest browser automation engine with improved reliability.
- **Stealth Mode**: Advanced stealth features to avoid detection during scraping or automation.
- **Proxy Support**: Easy configuration for rotating or static proxies.
- **Viewport Customization**: Specific tools for sizing and scaling browser viewports.

## How to Use
1. **Config**: Set `BROWSERBASE_API_KEY` and `BROWSERBASE_PROJECT_ID`.
2. **Launch**: `npx -y mcp-server-browserbase-local`.
3. **Automate**: Use natural language to describe browser-based tasks.

## Requirements
- **Runtime**: Node.js/NPM.
- **API Key**: Browserbase account required.
