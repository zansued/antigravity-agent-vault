# mcp-game-asset-gen

An MCP server for generating Three.js and game engine assets, including models, textures, and blueprints.

## Context
Use this skill when developing games or 3D web applications that require automated asset generation and Three.js component blueprinting.

## Key Features
- **Asset Generation**: Automates the creation of 3D assets for Three.js.
- **Engine Support**: Compatible with major game engines via standardized asset formats.
- **Texture/Material Matching**: Intelligently suggests and generates matching materials.
- **Blueprinting**: Generates Three.js boilerplate code for imported assets.

## How to Use
1. **Generate**: "Generate a Three.js-compatible 3D model of a [object]".
2. **Apply**: "Generate a realistic texture for this model".
3. **Draft**: "Create a Three.js scene blueprint for these assets".

## Requirements
- **Runtime**: Node.js/NPM.
- **3D Lib**: Three.js integration recommended.
