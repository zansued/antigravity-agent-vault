# flutter-prd-generator

A specialized extension for converting Product Requirement Documents (PRD) into production-ready Flutter development guides and boilerplate.

## Context
Use this skill when starting a new Flutter project or feature from a PRD, ensuring it follows Clean Architecture, BLoC patterns, and industry compliance (HIPAA, PCI-DSS, GDPR).

## Key Features
- **PRD Transformation**: Automatically generates 32+ development guides from a single PRD.
- **Compliance Ready**: Built-in support for healthcare (HIPAA) and fintech (PCI-DSS) security standards.
- **Architectural Enforcement**: Enforces Clean Architecture and BLoC state management patterns.
- **React-to-Flutter Refactoring**: Dedicated tools for migrating React components to Flutter.
- **Slash Commands**: Includes 21 comprehensive slash commands for rapid scaffolding and localization.

## How to Use
1. **Convert**: "Generate a Flutter project guide from the provided PRD".
2. **Localize**: "Add multi-language support to the generated Flutter app".
3. **Refactor**: "Convert the [Component].tsx to a Flutter BLoC widget".

## Requirements
- **Framework**: Flutter (latest stable).
- **Patterns**: BLoC, Clean Architecture.
