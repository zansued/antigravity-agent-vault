# rune-intelligence-vault

An MCP-based intelligence management system designed to "stop corporate amnesia" by assetizing intelligence.

## Context
Use this skill to manage team-wide intelligence, access shared knowledge vaults, and ensure long-term persistence of organizational knowledge.

## Key Features
- **Intelligence Assetization**: Converts raw information into persistent intelligence assets.
- **Rune-Vault**: Centralized secure repository for organizational knowledge.
- **enVector Integration**: Uses enVector Cloud for advanced vector-based indexing.
- **Multi-Agent Support**: Compatible with Claude Code, Codex, and Gemini CLI.

## How to Use
1. **Prerequisites**: Requires Rune-Vault access and enVector credentials.
2. **Setup**: Follow the team onboarding via `rune init`.
3. **Query**: Access shared assets via natural language queries within the vault.

## Requirements
- **Cloud**: enVector Cloud account.
- **Auth**: Team-level vault credentials.
