# community-apify-skill-collection

A community-maintained collection of specialized Apify agent skills, providing additional web scraping and automation capabilities beyond the core Apify toolset.

## Context
Use this skill for niche web automation tasks, community-contributed scrapers, or experimental Apify-based agent tools.

## Key Features
- **Community Scrapers**: Access to a wide range of community-built scrapers for specific platforms (social media, e-commerce, news).
- **Experimental Primitives**: Early-access or experimental tools for advanced web interaction patterns.
- **Cross-Platform Support**: Skills tailored for specific AI assistants and coding environments (Gemini CLI, Cursor, Windsurf).
- **Modular Integration**: Allows for dynamic loading of specific community skills as needed for complex workflows.

## How to Use
1. **Explore**: "List all available skills in the community Apify collection".
2. **Execute**: "Use the [CommunitySkillName] from the awesome-skills collection to [Action]".
3. **Combine**: "Integrate the community [SkillA] with the core [SkillB] to perform a multi-site data extraction".

## Requirements
- **Account**: Apify account and API token.
- **Runtime**: Gemini CLI + Awesome Apify Skills.
