# visual-explainer-enhanced

An enhanced version of the visual-explainer agent skill, utilizing nano-banana-2 for high-fidelity image generation and 3.1-pro-preview aesthetics.

## Context
Use this skill when you need to provide visual explanations of complex concepts, diagrams, or UI mockups with high aesthetic quality.

## Key Features
- **Enhanced Aesthetics**: Specifically tuned for Gemini 3.1-pro-preview to ensure "wow" factor visual outputs.
- **Nano-Banana Integration**: Direct integration with `nano-banana-2` for state-of-the-art image generation.
- **Visualization Commands**: Dedicated tools for generating architecture diagrams, flowcharts, and concept art.
- **Refined Outlooks**: Improved reasoning models for describing and justifying visual decisions.

## How to Use
1. **Explain**: "Visually explain the concept of [topic] using the enhanced explainer".
2. **Generate**: "Create a high-fidelity diagram of the current application's data flow".
3. **Refine**: "Improve the aesthetics of the previously generated visual".

## Requirements
- **Integration**: Requires `nano-banana-2` or compatible image generation MCP.
