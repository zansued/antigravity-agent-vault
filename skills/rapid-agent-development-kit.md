# rapid-agent-development-kit

A collection of specialized extensions and tools designed to accelerate the development, testing, and deployment cycles for autonomous agents.

## Context
Use this skill when bootstrapping new agent projects, implementing rapid prototyping workflows, or optimizing agentic interaction layers.

## Key Features
- **Project Scaffolding**: Rapidly initialize agentic project structures following best practices.
- **Development Productivity**: Tools and scripts focused on reducing latency between agent design and verification.
- **Modular Extensions**: Support for adding specialized capabilities to agents with minimal configuration.
- **Testing Integration**: Pre-configured hooks for validating agent behavior early in the development cycle.

## How to Use
1. **Bootstrap**: "Use the Rapid Agent Development kit to scaffold a new [AgentType] in the current directory".
2. **Configure**: "Apply the [ExtensionName] module to the current agent configuration".
3. **Optimize**: "Analyze the agent's interaction loop and apply rapid-dev performance tweaks".

## Requirements
- **Runtime**: Gemini CLI / Node.js.
- **Environment**: Optimized for fast-iteration development environments.
