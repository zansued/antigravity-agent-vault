# rocketchat-apps-generator

A Gemini CLI extension for rapid generation and scaffolding of Rocket.Chat applications.

## Context
Use this skill to bootstrap new Rocket.Chat apps with standard boilerplate and type-safe configurations.

## Key Features
- **App Scaffolding**: Generates the necessary file structure for official Rocket.Chat apps.
- **Type-Safety**: Ensures generated code follows the Rocket.Chat Apps-Engine TypeScript types.
- **Gemini Integration**: Seamlessly integrates with the Gemini CLI for interactive generation.
- **Modern Boilerplate**: Uses the latest Rocket.Chat app patterns (v2.x+).

## How to Use
1. **New App**: "Scaffold a new Rocket.Chat app named [AppName]".
2. **Add Feature**: "Add a slash command listener to my [AppName]".
3. **Configure**: "Generate the `manifest.json` for the current Rocket.Chat app".

## Requirements
- **Tools**: Rocket.Chat Apps-Engine CLI.
- **Language**: TypeScript.
