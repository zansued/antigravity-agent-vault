# boring-gemini

An agentic cognition support tool emphasizing rigorous workflows, active recall memory, and production-grade safety.

## Context
Use this skill for complex project planning, logic verification, and knowledge management where high-speed autonomy is required.

## Pillars of Autonomy
- **Cognitive Reasoning**: Advanced agentic loops for multi-step reasoning.
- **Active Recall**: Resilient memory system for long-term project understanding.
- **Shadow Mode**: Production-grade safety net for autonomous actions.
- **UV-Native**: High-speed execution using the `uv` ecosystem.

## Modes
- **Cyborg Mode**: Operates as an MCP server for human-in-the-loop assistance.
- **Autonomous Mode**: Runs as a full CLI agent for standalone tasks.

## How to Use
1. **Plan**: Start a project planning loop in Autonomous Mode.
2. **Verify**: Use logic verification tools during the development cycle.
3. **Audit**: Leverage the Safety Net for auditing changes before commit.

## Requirements
- **Ecosystem**: Works alongside Cursor, Copilot, and Gemini CLI.
