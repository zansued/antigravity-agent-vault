# rocketchat-apps-generator-gsoc

An AI-powered Rocket.Chat application generator prototype developed as part of GSoC 2026.

## Context
Use this skill to prototype and generate modern Rocket.Chat apps using an AI-native approach tailored for the latest Apps-Engine versions.

## Key Features
- **AI-Native Generation**: Uses advanced LLM prompts specifically designed for Rocket.Chat app architecture.
- **Prototyping Tools**: Rapidly creates functional prototypes of RC apps for testing and demonstration.
- **Modern Standards**: Adheres to the v2.x+ Rocket.Chat Apps-Engine standards.
- **GSoC Research**: Incorporates latest research on AI-assisted open-source development.

## How to Use
1. **Prototype**: "Generate a functional prototype for a Rocket.Chat [AppType] app".
2. **Generate**: "Scaffold the boilerplate for a new RC app based on [Requirements]".
3. **Research**: "Explain the architecture of the generated Rocket.Chat app".

## Requirements
- **Tools**: Rocket.Chat Apps-Engine CLI.
- **Timeline**: Optimized for GSoC 2026/Modern RC standards.
