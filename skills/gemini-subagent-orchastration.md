# gemini-subagent-orchastration

Configuration and orchestration for Gemini-based sub-agents.

## Context
Use this skill when managing complex task decomposition across multiple Gemini sub-agents. It provides a framework for configuring and coordinating agent interactions.

## Key Features
- **Config Management**: Centralized configuration for sub-agent roles and parameters.
- **Orchestration**: Patterns for delegating tasks and aggregating results from multiple agents.
- **Gemini Specific**: Optimized for use with the Gemini API and CLI extensions.

## How to Use
1. **Setup**: Define sub-agent configurations in the project structure.
2. **Orchestrate**: Use defined patterns to trigger and manage sub-agent loops.
