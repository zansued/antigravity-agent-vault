# magicline-api-extension

An extension providing deep context, structured rules, and implementation patterns for developing against the Magicline API.

## Context
Use this skill when developing integrations, features, or automation for the Magicline ecosystem to ensure compliance with its specific API patterns and constraints.

## Key Features
- **Structured Rules**: Enforces Magicline-specific coding standards and API usage limits.
- **Local Documentation**: Provides rapid access to Magicline API endpoints and data models without external lookups.
- **Implementation Patterns**: Includes pre-verified patterns for common Magicline tasks (e.g., member management, booking).
- **Context Injection**: Automatically injects relevant Magicline technical context into agent prompts.

## How to Use
1. **Query**: "List all member-related endpoints in the Magicline API".
2. **Implement**: "Create a member registration flow following Magicline patterns".
3. **Validate**: "Audit the current Magicline integration for common pitfalls".

## Requirements
- **API**: Access to Magicline developer credentials.
- **Engine**: Gemini CLI extension enabled.
