# mediationverse-r-dev

A comprehensive Gemini CLI extension for R package development with 33 agentic commands.

## Context
Use this skill for intensive R package development, including planning, session management, and CI/CD integration.

## Command Categories
- **Setup & Planning**: Initialize R projects and plan package structure.
- **Session Management**: Track and manage R development sessions.
- **Development**: Core R coding and package building tools.
- **Website & Docs**: Automated generation of pkgdown sites and documentation.
- **Git & CI/CD**: Specialized Git workflows for R/CRAN.
- **Quality & Review**: Deep analysis of R code quality and review cycles.

## How to Use
1. **Init**: "Initialize a new R package with mediationverse".
2. **Develop**: Use targeted commands for building and testing.
3. **Release**: Use the Git & CI/CD commands for CRAN-compliant releases.

## Requirements
- **Runtime**: R environment must be installed.
- **Platform**: Designed for Gemini CLI.
