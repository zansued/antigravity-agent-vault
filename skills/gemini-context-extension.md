# gemini-context-extension

Price and context window tracker for Gemini-based workflows, including repository analysis and wiki generation.

## Context
Use this skill when you need to monitor token usage, estimate API costs, compare Gemini models, or perform high-level repository analysis for documentation.

## Key Features
- **Context Tracker**: Real-time monitoring of the active context window.
- **Cost Estimator**: Accurate estimation of API costs based on token usage.
- **Repository Analyzer**: Scans projects to provide structural insights.
- **Wiki Generator**: Automatically generates project documentation/wikis.
- **Semantic Search**: (Phase 4) Advanced code search within the repository.

## How to Use
1. **Monitor**: "Show current context window and cost estimation".
2. **Analyze**: "Analyze this repository and generate a wiki".
3. **Compare**: "Compare performance/cost for Gemini 2.0 vs 1.5 for this task".

## Requirements
- **Runtime**: Node.js 18.x+.
- **Environment**: Gemini CLI.
