# go-microservice-guardrails

Reusable architecture specifications and guardrails for Go microservice development.

## Context
Use this skill when developing Go microservices to ensure compliance with architectural standards and best practices. It provides guardrails for Claude Code, OpenAI Codex, and Gemini CLI.

## Key Features
- **Architectural Specs**: Reusable patterns for Go-based microservices.
- **Guardrails**: Automated checks and guidance for maintaining code quality and consistency.
- **Multi-Platform Support**: Works with major AI coding assistants.

## How to Use
1. **Reference**: Ask the assistant to apply Go microservice guardrails during design or implementation.
2. **Review**: Use the guardrails to validate existing service structures.

## Requirements
- **Language**: Go.
- **Environment**: Compatible with Gemini CLI, Claude Code, or OpenAI Codex.
