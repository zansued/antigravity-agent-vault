# slidev-presentation-extension

A specialized extension for Gemini CLI designed for creating, managing, and presenting digital slides using the Slidev framework.

## Context
Use this skill to automate the creation of presentation slides, generate content based on outlines, and manage Slidev-based slide decks via natural language.

## Key Features
- **Natural Language Deck Creation**: "Create a 5-slide presentation about [Topic] using Slidev".
- **Slidev CLI Integration**: Direct access to Slidev commands for building, exporting, and serving presentations.
- **Content Generation**: Automatically generates markdown content for slides based on provided topics or research.
- **Theming Support**: Assisted configuration of Slidev themes and layout styles.

## How to Use
1. **Create**: "Scaffold a new Slidev project for a developer keynote".
2. **Update**: "Add a slide about [SubTopic] after the third slide".
3. **Export**: "Export the current Slidev presentation to a PDF for sharing".

## Requirements
- **Tools**: Slidev installed (`npm install -g @slidev/cli`).
- **Runtime**: Node.js/NPM + Gemini CLI.
