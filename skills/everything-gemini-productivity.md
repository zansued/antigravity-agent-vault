# everything-gemini-productivity

A comprehensive collection of Gemini CLI configurations, pre-built agents, and productivity-focused skills.

## Context
Use this skill to access a broad library of productivity configurations and pre-initialized agents for various development tasks.

## Key Features
- **Pre-built Agents**: Includes specialized agents for coding, documentation, and project management.
- **Custom Skills**: High-productivity skills for rapid development and context management.
- **MCP Server Library**: Pre-configured connections to various MCP servers.
- **Productivity Configs**: Optimized `.agent` and `.gemini` configurations.

## How to Use
1. **Setup**: Follow the rapid installation guide to sync configurations.
2. **Execute**: Use the provided slash commands for common tasks (e.g., `/refactor`, `/document`).
3. **Load**: "Load the productivity-vibe agent for the current session".

## Requirements
- **Environment**: Optimized for Gemini CLI and Antigravity.
- **Installation**: Requires the synchronization script for full feature set.
