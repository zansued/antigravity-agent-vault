# rocketchat-apps-generator-soroushrf

A specialized Rocket.Chat Apps Generator extension for Gemini CLI, providing tools for scaffolding and managing Rocket.Chat app projects.

## Context
Use this skill when building or managing Rocket.Chat apps to automate repetitive scaffolding tasks and ensure adherence to RC App engine standards.

## Key Features
- **App Scaffolding**: Rapidly generate the boilerplate for a new Rocket.Chat app.
- **Project Management**: Tools for organizing and updating RC app project structures.
- **Gemini CLI Integration**: Tailored for use within the Gemini CLI environment.
- **Best Practices**: Pre-configures apps following standard Rocket.Chat developer guidance.

## How to Use
1. **Scaffold**: "Generate a new Rocket.Chat app named [AppName] with basic UI hooks".
2. **Update**: "Update the project structure for the current RC app to match the latest version".
3. **Verify**: "Check the [AppName] project for compatibility with the RC App Engine".

## Requirements
- **Tools**: Rocket.Chat Apps CLI installed.
- **Runtime**: Gemini CLI.
