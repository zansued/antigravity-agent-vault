# chrome-devtools-specialist

An advanced Gemini CLI extension for deep integration with Chrome DevTools, providing agents with surgical control over browser debugging, performance profiling, and DOM manipulation.

## Context
Use this skill for complex frontend debugging, web performance optimization, or automated browser-based testing and interaction.

## Key Features
- **Surgical DOM Control**: Precise manipulation and querying of the DOM tree beyond simple selectors.
- **Performance Profiling**: Captures and analyzes performance traces, identifying bottlenecks and layout thrashing.
- **Network Auditing**: Deep inspection of network requests, headers, and payload data with filtering capabilities.
- **Console Orchestration**: Interacts with the browser console, executing scripts and capturing errors with full stack traces.
- **WebSocket Support**: Advanced connectivity options including WebSocket-based debugging for modern web apps.

## How to Use
1. **Debug**: "Open the Chrome DevTools session and locate the cause of the [ErrorType] in the [Component]".
2. **Profile**: "Record a performance trace of the [Action] and identify the longest running main-thread tasks".
3. **Intercept**: "Audit the network calls triggered by [Action] and verify that the [PayloadData] is correct".

## Requirements
- **Browser**: Chrome or Chromium-based browser with remote debugging enabled.
- **Runtime**: Gemini CLI + Chrome DevTools MCP Specialist.
