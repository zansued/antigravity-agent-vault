# AI Strategy & Architecture

Diretrizes para a construção de sistemas que utilizam Inteligência Artificial em seu núcleo (AI-First).

## Pilares de Design

### 1. Autonomia vs Controle
- Defina gatilhos claros para supervisão humana.
- Implemente 'Watchdogs' para monitorar o comportamento do agente.

### 2. Avaliação de Modelos
- Selecione modelos baseando-se em: Token Economics, Latency e Reasoning Depth.
- Use Sonnet para codificação complexa e Flash para tarefas de orquestração rápida.

### 3. Ecossistema de Skills
- Construa sistemas modulares onde cada capacidade é uma 'Skill' independente e plugável.
