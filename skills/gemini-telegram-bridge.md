# gemini-telegram-bridge

A comprehensive Telegram integration extension for Gemini CLI, allowing AI agents to fully interact with the Telegram messaging platform.

## Context
Use this skill when your agent needs to send messages, receive updates, or manage Telegram chats and media as part of its workflow.

## Key Features
- **Full Messaging Support**: Send and edit text messages, as well as delete messages.
- **Media Management**: Send photos and documents directly to chats.
- **Chat Interaction**: Tools for reading chat history, getting chat info, and managing callback queries.
- **Administrative Control**: Retrieve chat administrator lists and perform admin-level actions.
- **Native Extension**: Built specifically for the Gemini CLI ecosystem.

## How to Use
1. **Message**: "Send a Telegram message to the [DevelopmentGroup] with the latest build results".
2. **Edit**: "Edit the previous Telegram message in [ChatID] to reflect the updated status".
3. **Send Media**: "Upload the [SystemDiagram].png to the [ArchitectureChat] on Telegram".

## Requirements
- **API**: Requires a Telegram Bot Token.
- **Runtime**: Gemini CLI.
