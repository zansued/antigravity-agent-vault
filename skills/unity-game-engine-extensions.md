# unity-game-engine-extensions

A specialized extension for Unity game development, providing deep assistance for project structure, C# scripting, and component analysis.

## Context
Use this skill when developing Unity games to analyze project hierarchy, optimize C# scripts, and navigate the Unity Editor environment using AI.

## Key Features
- **Project Navigation**: Efficiently queries Unity project assets, folders, and hierarchy.
- **C# Scripting Assistance**: Provides specialized help for Unity-specific APIs (MonoBehaviour, ScriptableObjects, Physics, etc.).
- **Component Analysis**: Analyzes GameObject components and their relationships.
- **GameDev Workflow**: Optimizes the development loop within the Unity ecosystem.

## How to Use
1. **Analyze**: "Analyze the components attached to the [ObjectName] prefab".
2. **Script**: "Help me write a Unity C# script for a [FeatureName] using best practices".
3. **Navigate**: "Find all usages of the [AssetName] in the current Unity project".

## Requirements
- **Framework**: Unity Engine (latest stable recommended).
- **Language**: C#.
