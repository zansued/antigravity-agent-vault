# gemini-coding-mentor

An extension that transforms Gemini CLI into a socratic coding mentor, focusing on guidance and codebase navigation teaching.

## Context
Use this skill when you want the agent to act as a mentor rather than just an executor, providing guidance and forcing the user to think through problems.

## Key Features
- **Socratic Guidance**: Uses "Mentor Mode" to ask leading questions and guide the user toward solutions.
- **Navigation Teaching**: Teaches the user how to navigate and understand complex codebases.
- **Pre-flight Checks**: Automated checks before executing significant changes to ensure user understanding.
- **Experimental Learning**: Optimized for educational and collaborative development settings.

## How to Use
1. **Enable**: "Turn on Mentor Mode for the current project".
2. **Guide**: "Guide me through fixing the bug in the authentication module using socratic methods".
3. **Verify**: The mentor will ask questions to verify your understanding before proceeding.

## Requirements
- **Runtime**: Gemini CLI.
- **Phase**: Recommended for the PLANNING and DESIGN phases.
