# gitlab-gemini-extension

Official/Community-supported extension for Gemini CLI to interact with GitLab resources (Issues, MRs, Pipelines).

## Context
Use this skill when working in a GitLab-centric ecosystem to manage issues, merge requests, and CI/CD pipelines.

## Key Features
- **Issue Management**: Create, list, and update GitLab issues.
- **Merge Requests (MRs)**: Tools for MR review, creation, and approval.
- **Pipeline Control**: Trigger and monitor GitLab CI/CD pipelines.
- **Repository Interaction**: Seamless integration with GitLab git workflows.

## How to Use
1. **MR Review**: "Review my last merge request on [project]".
2. **Issue Tracking**: "Create an issue for the bug found in the production pipeline".
3. **Pipelines**: "What is the status of the current deployment pipeline?".

## Requirements
- **Auth**: GitLab Personal Access Token (PAT).
- **Environment**: Gemini CLI.
