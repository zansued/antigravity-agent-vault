# learnship

An agentic engineering framework focused on intentional building, project memory, and lifecycle management.

## Context
Use this skill for complex project engineering where intentional design, persistent memory, and rigorous milestone tracking are critical.

## Key Concepts
- **The Phase Loop**: A structured development cycle (Discovery -> Design -> Execution -> Verification).
- **AGENTS.md**: Persistent project memory file used to maintain context across long sessions.
- **Decision Intelligence**: Tools for documenting and retrieving key architectural decisions.
- **Milestone Intelligence**: Automated tracking of project milestones and UAT status.

## Workflow Triggers
1. **Init**: `npx learnship init` to establish project memory.
2. **Engineer**: Use agentic loops for greenfield or brownfield tasks.
3. **Verify**: Automated UAT and Gap analysis after each milestone.

## Requirements
- **Framework**: Built on top of Gemini CLI and standard agentic patterns.
