# augment-code-context-engine

The Context Engine MCP by Augment Code, designed to provide deep contextual awareness and repository-level intelligence to AI coding assistants.

## Context
Use this skill when you need high-fidelity context about a project's architecture, dependency graphs, or cross-file relationships.

## Key Features
- **Deep Contextual Engine**: A specialized engine that indexes and provides relevant code context on demand.
- **Repository Awareness**: Maps the entire repository to provide cross-file understanding and dependency analysis.
- **Integration Primitives**: Directly plugs into Gemini CLI and other MCP-compatible coding platforms.
- **Intelligent Retrieval**: Uses advanced indexing to retrieve the most relevant code snippets for a given task.
- **Architectural Mapping**: Assists in understanding the high-level design of complex software systems.

## How to Use
1. **Index**: "Build the context index for the current repository using the Augment Code engine".
2. **Query**: "Ask the context engine to explain the relationship between [ModuleA] and [ModuleB]".
3. **Retrieve**: "Retrieve all relevant architectural patterns found in the [ProjectDir]".

## Requirements
- **Runtime**: Gemini CLI + Augment Code Context Engine MCP.
- **Integration**: Designed for large-scale enterprise codebases.
