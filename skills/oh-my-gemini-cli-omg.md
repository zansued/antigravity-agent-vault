# oh-my-gemini-cli-omg

A comprehensive workflow pack for Gemini CLI powered by context-engineering and multi-agent team assembly.

## Context
Use this skill for complex, team-oriented projects that require dynamic agent assembly, strict workspace hygiene, and advanced context engineering.

## Key Features
- **Dynamic Team Assembly**: Assembles a tailored "team" of sub-agents based on task requirements.
- **Context Engineering**: Advanced techniques for maintaining and optimizing project context layers.
- **Workspace Hygiene**: Automated hooks for checking workspace status and maintaining "Symmetry".
- **Notification Routing**: Intelligent routing of agent updates and taskboard status.
- **Automatic Usage Monitor**: Tracks and reports token/cost usage after each agent operation.

## Sovereign Commands (Chronos-O)
- `/galta`: [G]lobal [A]sync [L]edger [T]racking [A]gent. Status e Sincronia.
- `/chronos`: Ativação do Motor Omega (Alta Velocidade).
- `/mapa`: Sincronia com Mapa Mental (Spatial Architect).
- `/coliseu`: Debate Exponencial (20 Agentes).
- `/dna`: Destilação e Gravação de Aprendizado.
- `/lista`: Exibe TODOS os comandos e sub-agentes.
- `/pulse`: Limpeza técnica e destilação de ouro.

## Requirements
- **Runtime**: Node.js/NPM.
- **Integration**: Full integration with Gemini CLI hooks.
Proxy: v0.4.0+ recommended.
