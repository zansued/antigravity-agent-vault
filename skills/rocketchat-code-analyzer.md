# rocketchat-code-analyzer

An MCP-based code analyzer specifically designed for the Rocket.Chat ecosystem and codebase.

## Context
Use this skill when developing for or searching through the Rocket.Chat repository to understand session lifecycles, API endpoints, and internal logic.

## Key Features
- **Deep Analysis**: Specifically tuned for the unique patterns and architecture of Rocket.Chat.
- **Session Lifecycle Monitoring**: Special tools for analyzing how sessions are handled in RC.
- **MCP Toolset**: Provides direct analysis tools via the Model Context Protocol.
- **Research Oriented**: Optimized for GSoC and open-source contributors to Rocket.Chat.

## How to Use
1. **Analyze**: "Analyze the authentication session lifecycle in the Rocket.Chat codebase".
2. **Query**: "Find all usages of [RC_API_METHOD] and explain the implementation".
3. **Audit**: "Perform a code quality audit on the new Rocket.Chat app module".

## Requirements
- **Target**: Rocket.Chat codebase (GitHub clone).
- **Runtime**: Node.js/NPM.
