# youtube-to-docs

A tool for converting YouTube video content into structured documentation, summaries, and multi-media assets.

## Context
Use this skill when you need to extract knowledge from video tutorials, presentations, or meetings hosted on YouTube and convert them into searchable documentation.

## Key Features
- **Structured Docs**: Automatically generates markdown documentation from video transcripts.
- **Summarization**: Creates concise summaries of key points and takeaways.
- **Asset Extraction**: Extracts audio and visual assets for easier discovery.
- **Knowledge Discovery**: Facilitates rapid learning from video-based resources.

## How to Use
1. **Extract**: "Connect to YouTube video [URL] and generate documentation".
2. **Summarize**: "Summarize the key architectural decisions from this video".
3. **Index**: Use the extracted docs as part of your project's local knowledge base.

## Requirements
- **Integration**: Requires YouTube Data API access or equivalent scraping capabilities.
