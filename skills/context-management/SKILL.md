# Context Management (Metatron Memory & CELF-8)

Garante que o Metatron mantenha a precisão técnica suprema mesmo em conversas extensas, utilizando o framework CELF de 8 camadas para evitar o efeito 'lost-in-the-middle'.

## Estratégias de Preservação e Engenharia

### 1. Camadas CELF (Contextual Engineering)
- Aplique rigorosamente as camadas de **L0 (Constituição)** a **L7 (Delegação)** em cada requisição complexa.
- Priorize a **Identidade (L1)** e o **Domínio (L2)** como filtros constantes.

### 2. Snapshots e Destilação Apex
- Use **Token Sculpting** para remover metadados irrelevantes de logs e traces.
- Salve snapshots semânticos no Ledger (`geminicli_knowledge_nodes`) usando **Níveis Epistêmicos (AXIOMA, FATO, etc.)**.

### 3. Recuperação por Tiers
- Recupere **AXIOMAS** e **FATOS** para execução imediata.
- Use **HEURÍSTICAS** para exploração criativa e resolução de bugs complexos.
